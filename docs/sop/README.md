## Writing rules

- Start with the user goal, not the tax section.
- Use short Hindi-English sentences where it helps clarity.
- Every value the user copies must have a source: Form 16, AIS, 26AS, CAMS, broker P&L, or manual confirmation.
- Never say we e-file. We prepare and guide; the user submits on `incometax.gov.in`.
