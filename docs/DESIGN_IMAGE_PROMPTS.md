# Design Image Prompts (Indian Taxpayer Context)

Use these prompts for generated visuals across the landing and learning surfaces. Keep the style coherent: minimal flat illustration, clean geometry, limited palette, modern SaaS look, no stock-photo realism.

## Global style guidance (apply to all prompts)

- Style: minimal flat illustration, soft shadows, rounded UI cards, clean whitespace.
- Palette: whites and cool blues as base, optional saffron accent highlights, subtle Indian context motifs.
- Characters: Indian professionals and families in contemporary attire, confident and calm.
- Avoid: clutter, caricature stereotypes, over-decorative flags, cliché tax imagery, photo-real stock look.

## 1) Hero visual — AI filing companion (right side)

- **Prompt:** "Minimal flat illustration of an Indian salaried professional seated with a laptop, using a clean AI tax companion dashboard. Dashboard should hint at incometax.gov.in style context without logos: simple form fields, AY badge, rupee amounts, and checklist cards. Add subtle trust cues (shield/check), modern workspace, white background with blue accents and tiny saffron highlight dots."
- **Aspect ratio:** `3:2` (desktop hero card) and crop-safe to `4:3`.
- **Placement:** Home hero right column (`/`), above or beside regime compare card.

## 2) Filing season AY badge accent

- **Prompt:** "Minimal vector badge set for Indian filing season, showing 'AY 2026-27' style labels, with clean pill shapes, tiny calendar icon, and muted blue + saffron accents. Keep it elegant and UI-ready, no heavy textures."
- **Aspect ratio:** `1:1` and `4:1` variants.
- **Placement:** Hero micro-accent, section chips, timeline headers.

## 3) Trust and security section visual

- **Prompt:** "Flat illustration of secure Indian tax filing: shield icon, encrypted document flow, masked personal data card, and calm user confirmation state. Use clean blue-white interface cards with subtle saffron accent dots, minimal visual noise."
- **Aspect ratio:** `16:9`.
- **Placement:** Security/trust strip and reassurance sections.

## 4) Form 16 upload moment

- **Prompt:** "Modern flat scene of an Indian taxpayer uploading Form 16 into a tax app. Show drag-and-drop card, PDF icon labeled 'Form 16', progress indicator, and success check. Keep composition minimal, with soft blue-gray surfaces and tiny saffron highlights."
- **Aspect ratio:** `4:3`.
- **Placement:** Upload/import section cards and onboarding steps.

## 5) Companion wizard walkthrough

- **Prompt:** "Minimal multi-step wizard illustration for ITR filing companion: step cards for profile, income, deductions, review, submit. Include one Indian user avatar and clean directional flow arrows. UI-first style, generous whitespace, flat vector look."
- **Aspect ratio:** `16:10`.
- **Placement:** Companion mode explainer and feature walkthrough section.

## 6) Glossary learning visual

- **Prompt:** "Simple educational illustration for tax glossary learning: searchable terms panel, highlighted keywords like TDS, 80C, HRA, and plain-language explanation cards. Indian learner persona, approachable and clean, white + blue palette with optional saffron micro accents."
- **Aspect ratio:** `3:2`.
- **Placement:** Glossary page hero header and empty/search states.

## 7) Empty state — no search results

- **Prompt:** "Minimal empty-state illustration for no glossary results in a tax app: magnifier over clean cards, muted text placeholders, friendly but professional Indian-context UI style. Keep light, airy, and non-distracting."
- **Aspect ratio:** `1:1`.
- **Placement:** Glossary no-result state component.

## 8) Regime compare support visual

- **Prompt:** "Flat illustration of Old vs New tax regime comparison with two balanced cards, rupee values, and a subtle 'recommended' check mark. Clean fintech UI style, Indian context through ₹ formatting and AY tag, restrained color usage."
- **Aspect ratio:** `5:4`.
- **Placement:** Hero/regime compare support blocks, blog explainers.
